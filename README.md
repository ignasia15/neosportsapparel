# Website Profile — PHP Native

Website profile sederhana berbasis web yang dibangun menggunakan PHP native dan CSS murni. Project ini dikerjakan saat Praktik Kerja Lapangan (PKL) dan sekaligus dijadikan sebagai Tugas Akhir SMK.

## Teknologi yang Digunakan
- PHP Native
- CSS3

## Fitur
- Halaman profil yang menampilkan informasi secara terstruktur
- Tampilan responsif menggunakan CSS

## Tujuan Project
Dibangun sebagai project PKL sekaligus Tugas Akhir dengan tujuan menerapkan dasar-dasar pemrograman web menggunakan PHP dan CSS tanpa framework tambahan.

---
> Project ini merupakan bagian dari pembelajaran selama masa PKL dan Tugas Akhir SMK Rekayasa Perangkat Lunak.
